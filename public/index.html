脚本注入教程 - 支持桌面端与移动端

📦 方法一：油猴脚本（推荐）

💻 桌面端 🤖 安卓

适用于 Chrome、Firefox、Edge、Safari 等浏览器

步骤：

  1. 安装油猴扩展

    根据你的浏览器选择对应的扩展：

    Chrome/Edge: Tampermonkey

    Firefox: Tampermonkey

    Safari: Tampermonkey

    安卓: 使用 Kiwi Browser 或 Firefox 安装 Tampermonkey

  2. 安装脚本

    点击下方按钮直接安装：

    📥 安装脚本
  3. 访问游戏

    打开游戏网站，脚本将自动运行

🔖 方法二：书签注入

📱 iOS 🤖 安卓 💻 桌面端

适用于所有平台，无需安装扩展

步骤：

  1. 复制注入代码

    点击下方按钮复制代码：

    javascript:(_=>{const script = document.createElement("script");script.src = `https://inkerbot.github.io/bc-mod-manager/main.js`;document.head.appendChild(script);})()
  2. 创建书签

    iOS Safari:

    随便收藏一个网页

    进入书签管理，编辑刚才的书签

    将书签名称改为 "BC Mod Manager"

    将书签地址替换为上面复制的代码

    安卓/桌面端:

    添加新书签（任意网址）

    编辑书签，将地址替换为上面的代码

  3. 使用书签

    在游戏页面中，点击刚才创建的书签即可注入脚本

💡 提示： 每次刷新页面后需要重新点击书签来注入脚本

⌨️ 方法三：控制台注入

💻 桌面端

适用于开发者或临时使用

步骤：

  1. 打开开发者工具

    按 F12 或 Ctrl+Shift+I (Mac: Cmd+Option+I)

  2. 切换到 Console 标签
  3. 粘贴并运行代码
    (_=>{ const script = document.createElement("script"); script.src = "https://inkerbot.github.io/bc-mod-manager/main.js"; document.head.appendChild(script); })()
  4. 按回车执行

    脚本将立即加载并运行

⚠️ 注意： 每次刷新页面后需要重新执行代码

❓ 常见问题

Q: 脚本无法加载？

A: 请检查：

  • 网络连接是否正常
  • 是否在游戏页面中运行
  • 浏览器是否阻止了脚本加载（检查控制台错误信息）

Q: iOS Safari 书签无法使用？

A: 确保：

  • 代码以 javascript: 开头
  • 代码完整复制，没有遗漏
  • 在游戏页面中点击书签，而不是在新标签页

Q: 安卓手机如何使用？

A: 推荐方案：

  • 方案1： 使用 Kiwi Browser + Tampermonkey（最稳定）
  • 方案2： 使用 Firefox + Tampermonkey
  • 方案3： 使用书签注入（任意浏览器）

Q: 脚本安全吗？

A: 本脚本是开源的，你可以在 GitHub 上查看完整源代码。脚本仅在本地运行，不会上传任何数据。
